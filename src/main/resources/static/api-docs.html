Image Manipulation Service API Documentation

Available Endpoints

1. Load Image

Method POST
URL /api/load
Parameters file (MultipartFile)
Description Upload an image file

2. Save Image

Method GET
URL /api/save
Parameters imageName (String),
destination (String)
Description Saves processed image at the specified location and sends it back

3. Brighten Image

Method POST
URL /api/brighten
Parameters imageName (String),
scale (int)
Description Adjust the brightness of an image

4. Flip Image

Method POST
URL /api/flip
Parameters imageName (String),
orientation (String: 'vertical' or 'horizontal')
Description Flip an image vertically or horizontally

5. Greyscale Image

Method POST
URL /api/greyscale
Parameters imageName (String),
component (String: 'value-component', 'luma-component', 'intensity-component', 'red-component', 'green-component', 'blue-component')
Description Convert an image to greyscale using the specified component

6. RGB Split

Method POST
URL /api/rgb-split
Parameters imageName (String)
Description Split an image into its RGB components

7. RGB Combine

Method POST
URL /api/rgb-combine
Parameters redImage (String),
greenImage (String),
blueImage (String)
Description Combine RGB components into a single image

8. Blur Image

Method POST
URL /api/blur
Parameters imageName (String)
Description Apply a blur filter to an image

9. Sharpen Image

Method POST
URL /api/sharpen
Parameters imageName (String)
Description Apply a sharpen filter to an image

10. Sepia Filter

Method POST
URL /api/sepia
Parameters imageName (String)
Description Apply a sepia filter to an image

11. Dither Image

Method POST
URL /api/dither
Parameters imageName (String)
Description Apply dithering to an image

Note: For all operations except 'load', the 'imageName' parameter refers to the name returned by the load operation or a previous processing operation.
